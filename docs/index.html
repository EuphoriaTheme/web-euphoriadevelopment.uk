Quick Start Path

  1. Read Licensing and claim your key.
  2. Install resource package using Blueprint.
  3. If using Player Listing, configure Game API.
  4. Apply setup extensions (Refresh Theme, Translations, Resource Alerts).
  5. Review legal and usage limits before production rollout.

Most support issues are caused by missing license activation, stale CDN cache, or API URL/CORS mismatch.
