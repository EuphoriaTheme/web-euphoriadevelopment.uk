Euphoria Theme

Theme Customiser

Core Theme Settings

Configure primary color, logo URL, favicon URL, footer text (HTML supported), panel background, cookie banner, and navigation visibility.

Visual Modules

  • Login background + logo visibility
  • Loading screen background/logo
  • Seasonal effects (Christmas/Easter/Halloween)
  • Announcements (type, icon, text)
  • Server card backgrounds + glow/status accents

External Links + API URL

Button links can point to Discord, Store, and Status pages. API URL can be changed to your own Game API endpoint.

After changing visual/API settings, use Refresh Theme and clear edge cache if changes do not appear.
