Before You Start

  • Use hosted Game API, or self-host your own API.
  • Confirm API endpoint is reachable from panel.
  • Open a game server and navigate to Players section.

Page Controls

  • Refresh: run a new API query
  • Select Game: pick game profile
  • Custom Query Port: override server query port
  • Save Port: persist custom port
  • Reset to Default: clear override

If no players are shown, verify query port and check firewall rules for query traffic on your game server.
