Alert Configuration

  1. Open Blueprint Extensions and select Resource Alerts.
  2. Set CPU/memory/disk alert thresholds.
  3. Configure action button URL for incident response.

Tuning Strategy

Start with broader thresholds to prevent noisy alerts, then tighten as real load patterns become clear.

Too-aggressive limits can create alert fatigue and reduce response quality during real incidents.
